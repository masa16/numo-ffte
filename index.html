FFTE Ruby wrapper with Numo::NArray

FFTE

  • is a package to compute Discrete Fourier Transforms of 1-, 2- and 3- dimensional sequences of length (2^p)(3^q)(5^r), developed by Prof. Daisuke Takahashi at University of Tsukuba.
  • Visit FFTE web site

Ruby wrapper

  • GitHub site
  • Tentative API Document

Installation

  • FORTRAN compiler is required to build FFTE (gfortran, etc).
  • Install Numo::NArray
  • Install Numo::FFTE:
$ git clone git://github.com/masa16/numo-ffte.git
$ cd numo-ffte
$ rake build
$ gem install pkg/numo-ffte-*.gem
