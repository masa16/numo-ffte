# Ruby wrapper of FFTE

## FFTE
* A package to compute Discrete Fourier Transforms of
  1-, 2- and 3- dimensional sequences of length (2^p)*(3^q)*(5^r).
* Visit [FFTE web site](http://www.ffte.jp/)
